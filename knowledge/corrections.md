# Corrections - журнал фидбека владельца скилла

Append-only. Сюда дописываются правила из живого фидбека на конкретные тексты. Скилл читает этот файл в фазе 1 (аудит); правила отсюда сильнее дефолтов справочника, но никогда не отменяют жёсткие запреты.

Формат записи:

```
## YYYY-MM-DD - <тема>

Фидбек: <что не понравилось в конкретном тексте>

Правило: <конкретное действие для редактора>

Область: <тип текстов, к которым применимо; «все» по умолчанию>
```
