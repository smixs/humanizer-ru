# Corrections - журнал фидбека владельца скилла

Append-only. Сюда дописываются правила из живого фидбека на конкретные тексты. Скилл читает этот файл в фазе 1 (аудит); правила отсюда сильнее дефолтов справочника, но никогда не отменяют жёсткие запреты.

Формат записи:

```
## YYYY-MM-DD - <тема>

Фидбек: <что не понравилось в конкретном тексте>

Правило: <конкретное действие для редактора>

Область: <тип текстов, к которым применимо; «все» по умолчанию>
```

## Записи

## 2026-07-21 - самоописание процесса в README/changelog

Фидбек: в README скилла появилась фраза «Скилл впервые улучшали как нейросеть тренируют» и списки «Добавлено» с отчётом о проделанной работе. Владелец: «это же и есть humanizer skill - перестань писать как машина».

Правило: в README, changelog и анонсах не описывать процесс работы над текстом и не хвастаться методологией. Писать только то, что изменилось для читателя, языком ELI16. Секции «что мы сделали» - кандидаты на удаление целиком: история живёт в git.

Область: README, changelog, анонсы, посты о релизах
