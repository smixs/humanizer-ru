# Быстрый сканер слов-маркеров

Компактный чеклист для быстрого сканирования текста. Нашёл слово из списка - проверяй контекст и исправляй.

## Группа A - Критические (всегда исправлять)

```
является, представляет собой, выступает в роли, служит основой,
носит характер, знаменует собой, символизирует,
осуществлять, проводить работу, оказывать помощь,
принимать участие, вести борьбу, производить расчёт,
давать оценку, совершать ошибку,
«не просто», «не только... но и», «это не просто»,
«надеюсь, это поможет», «буду рад помочь», «если у вас есть вопросы»,
«отличный вопрос!», «замечательное наблюдение!»
```

## Группа B - Высокие (почти всегда)

```
ключевой, важнейший, решающий, поворотный,
демонстрирует, отражает, способствует, обеспечивает,
содействует, формирует, свидетельствует, подчёркивает,
«важно отметить», «следует подчеркнуть», «необходимо учитывать»,
«стоит обратить внимание», «нельзя не упомянуть»,
знаменует собой ключевой этап, символизирует приверженность,
вносит вклад в, закладывает основу для,
подчёркивая..., обеспечивая..., символизируя..., способствуя...,
было принято, выполняется, осуществляется (пассив)
```

## Группа C - Средние (по контексту)

```
может похвастаться, яркий, богатый, глубокий,
захватывающий дух, потрясающий, уникальный, новаторский,
расположенный в, в сердце,
по мнению экспертов, аналитики отмечают, ряд источников,
«от X до Y» (проверить шкалу),
возможно можно предположить, в определённой степени,
имплементировать, дедлайн, консенсус, фидбек, коллаборация
```

## Группа D - Стилистические (для голоса)

```
протагонист/главный герой/центральный персонаж (synonym cycling),
«несмотря на... сталкивается с вызовами»,
«будущее выглядит ярким», «впереди захватывающие времена»,
«путь к совершенству», «продолжает процветать»,
эмодзи в заголовках, **жирный** на каждом слове,
«в контексте вышеизложенного»
```

## Запрещённые конструкции (мгновенный стоп)

```
«не просто X, а Y»          → разбить на два утверждения
«не только X, но и Y»       → разбить на два утверждения
«это не просто...»           → сказать прямо
«—» (длинное тире)           → заменить на «-»
двоеточия после вводных      → убрать или перестроить
риторические вопросы          → заменить утверждением
```
